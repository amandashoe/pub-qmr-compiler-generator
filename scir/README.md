## Installation
1. `pip install -r requirements.txt`  
2. Clone [Stim](https://github.com/quantumlib/Stim)
3. Follow installation instructions in `Stim/glue/lattice_surgery/README.md`
4. [Optional] Install [glTF Tools VS Code extension](https://marketplace.visualstudio.com/items?itemName=cesium.gltf-vscode) for viewing pipe diagrams.
